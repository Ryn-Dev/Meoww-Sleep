Infrastruktur & Keamanan

Istilah Teknis Lanjutan

Empat komponen vital yang membuat internet bekerja dengan aman dan terarah.

IP Address

Alamat unik setiap komputer (KTP Digital). Contoh: 192.168.1.1

DNS

Buku telepon internet. Menerjemahkan nama domain ke IP Address.

SSL / TLS

Sertifikat keamanan yang mengenkripsi data (Gembok hijau di browser).

Cookie

Data kecil yang disimpan browser untuk mengingat user (Login, Keranjang).

Bagaimana DNS Bekerja?

1

User Request

Kamu mengetik google.com di browser.

2

DNS Lookup

Browser bertanya ke DNS Server:
"Di mana rumah google.com?"

3

IP Found

DNS menjawab: "Alamatnya di 142.250.1.1"

4

Connection

Browser terhubung ke server Google dan menampilkan website.
