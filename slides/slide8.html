Infrastructure

Istilah Data & Server

Di mana website sebenarnya "hidup" dan bagaimana data disimpan agar bisa diakses oleh semua orang.

Storage

Database

Gudang penyimpanan data terstruktur. Ibarat lemari arsip raksasa yang menyimpan info user, produk, dll.

Computer

Server

Komputer super kuat yang hidup 24 jam untuk melayani permintaan (request) data dari browser kamu.

Network

Cloud

Jaringan server global di internet. Data tidak di satu tempat fisik, bisa diakses dari mana saja & kapan saja.

Space

Hosting

Tempat menyewa "lahan" di server agar file website kamu bisa online dan dikunjungi orang lain.
